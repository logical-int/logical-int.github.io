## How to make a GitHub hosted website

1. Make a GitHub account or log in
2. Create a new **public** repository and name it yourusername.github.io (this domain will not reset or expire, as long as your username remains the same)
3. Upload your files (index.html, sub-pages, etc)
4. Visit your domain: https://www.replacemewithyourusername.github.io
